# 🏥 병원 경영진단 시스템

## 개요
의원급 의료기관의 경영 전반을 진단하고 AI 기반 전략 방향을 제시하는 Streamlit 웹 애플리케이션입니다.

---

## 📁 파일 구성

```
hospital_diagnosis/
├── app.py                    # 메인 Streamlit 앱 (핵심 파일)
├── create_template.py        # 엑셀 입력 양식 생성기
├── requirements.txt          # Python 패키지 목록
└── README.md                 # 본 파일
```

---

## 🚀 설치 및 실행

### 1. Python 환경 준비 (Python 3.9 이상 권장)

```bash
# 가상환경 생성 (선택)
python -m venv venv
source venv/bin/activate        # macOS/Linux
venv\Scripts\activate           # Windows
```

### 2. 패키지 설치

```bash
pip install -r requirements.txt
```

### 3. 앱 실행

```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 접속

## 📊 데이터 입력 방법

### 방법 A: 샘플 데이터 (즉시 체험)
- 사이드바에서 **"샘플 데이터 사용"** 체크
- **"분석 실행"** 버튼 클릭

### 방법 B: 실제 데이터 입력
1. 엑셀 템플릿 생성:
   ```bash
   python create_template.py
   ```
2. 생성된 `병원경영진단_입력양식.xlsx` 파일 작성
3. 사이드바에서 파일 업로드 후 분석 실행

## 🤖 AI API 설정

### Claude (Anthropic)
- https://console.anthropic.com 에서 API 키 발급
- 모델: claude-opus-4-5 사용

### OpenAI (GPT-4o)
- https://platform.openai.com 에서 API 키 발급
- 모델: gpt-4o 사용

### Google Gemini
- https://aistudio.google.com 에서 API 키 발급
- 모델: gemini-1.5-pro 사용

> ⚠️ API 키는 사이드바에서 입력하며, 세션 내에서만 사용되고 저장되지 않습니다.

## 📈 분석 항목

1. **인건비 분석** - 기본급/4대보험/인센티브/퇴직충당금, 인건비율 평가
2. **고정비 분석** - 임차료/렌탈/유지보수/전기/수도 등 항목별 분석
3. **소모품/약제 분석** - 의료소모품/약제비/장비 구매비 분석
4. **매출 분석** - 급여/비급여 구성, 성장률, 1인당 처방금액
5. **원무 분석** - 신환/재진 비율, 내원횟수, 주요 상병
6. **수익성 분석** - 영업이익률, 손익 구조, 개선 시뮬레이션
7. **AI 심층 진단** - Claude/GPT-4o/Gemini 전문 진단 보고서
8. **종합 보고서** - 경영 현황, 개선 과제, 전략 로드맵

## 🔧 커스터마이징

### 벤치마크 수치 조정
`app.py` 내 `# 업종 벤치마크` 주석 부분에서 업종 평균값 수정 가능

### AI 프롬프트 수정
`build_ai_prompt()` 함수에서 AI에게 전달하는 분석 요청 수정 가능

### 점수 기준 조정
`calc_score()` 함수와 각 `thresholds` 파라미터에서 점수 기준 조정 가능
